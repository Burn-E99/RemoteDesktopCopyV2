About RDCP×

Remote Desktop CoPy (RDCP) is a free tool providing private text documents to copy and paste from. The oddly specific name comes from my initial use when this was crappily built in PHP and made for only one user with only one copy-paste drop zone. The new V2 you are currently on supports theoretically infinite documents.
